<p align="center">Welcome to the repository for my personal blog, where I share insights, projects, and thoughts. Dive in, explore, and get inspired!</p>

## ✨ Features

- 🖥️ **UI/UX Design**: Elegant and user-friendly interface.
- ⚡️ **Responsive Design**: Seamless experience across all devices.
- 🌗 **Dark-Light Mode**: Switch between dark and light themes.
- 👨‍💼 **About Section**: Overview of my background and career journey.
- 🛠️ **Project Showcase**: Detailed descriptions of my key projects.
- ✍️ **Blog**: Insights, tutorials, and personal experiences.
- 🎨 **Design Portfolio**: Showcasing my UI/UX design work.
- 💼 **Services**: Offering web development and design consulting.
- ✍🏼 **Guestbook**: Leave your thoughts and comments.

## 🔧 Requirements

- **Node.js** `>=20.11.0` (with [Corepack](https://nodejs.org/api/corepack.html) enabled)
- **npm** `>=9.x`
- (Optional) **Vue Developer Tools** [Chrome Extension](https://microsoftedge.microsoft.com/addons/detail/vuejs-devtools/olofadcdnkkjdfgjcmjaadnlehnnihnl)

## 🚀 Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/IsHereZahin/isherezahin.me.git


2. Navigate into the project directory::
   ```sh
   cd isherezahin.me

3. Install dependencies:
   ```sh
   npm install

4. Start the development server:
   ```sh
   npm run dev
