# Contributing

Thank you for your interest in contributing! Contributions make this project better and are always welcome.

## How to Contribute

1. **Fork the repository** to your own account.
2. **Clone your fork** locally:

   ```bash
   git clone https://github.com/your-username/isherezahin.me.git
   cd isherezahin.me
   ```
3. **Create a new branch** using your name for the branch:

   ```bash
   git checkout -b your-username
   ```
4. **Before making a Pull Request**, make sure your branch is up-to-date with the `master` branch of the main repository:

   ```bash
   git pull origin master
   ```
5. **Make your changes** and ensure the code is clean and readable.
6. **Commit your changes** with a clear message:

   ```bash
   git commit -m "Add feature X / Fix issue Y"
   ```
7. **Push your branch** to your fork:

   ```bash
   git push origin your-username
   ```
8. **Open a Pull Request** to the `master` branch of the main repository:
   [https://github.com/IsHereZahin/isherezahin.me](https://github.com/IsHereZahin/isherezahin.me)

## Reporting Issues

If you find a bug or have a suggestion, please open an **issue** describing it. Include steps to reproduce, screenshots, or any relevant details.

## Guidelines

* Keep your code readable and maintainable.
* Follow the existing coding style.
* Be respectful in discussions and comments.
* Large changes or new features may be discussed before implementing.

By contributing, you agree to follow the [Community Guidelines](CODE_OF_CONDUCT.md) to ensure a welcoming environment for everyone.
